# ipeds-db
Build a Postgres DB from IPEDS Access files

Right now it uses the 2014-15 Provisional data from https://nces.ed.gov/ipeds/Section/accessdatabase/

## Requirements
* posix operating system (linux or mac os, that sort of thing)
* mdbtools
* csvkit
* postgres
* wget
* make

## To use

```bash
make
make clean
```
